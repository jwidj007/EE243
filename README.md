# EE243
Final Project EE243

**Paper:** 3D Human Reconstruction in the Wild with Synthetic Data Using Generative Models (HumanWild)

**Task:** Qualitative evaluation of HMR2.0 regressor trained on HumanWild dataset. 
Test images were curated to probe three failure modes: extreme poses, hand/face label noise, and lower limb orientation errors.

**Results:** Full image results available at [Google Drive](https://drive.google.com/drive/folders/1TBjg3Oca1foHxhRtn3D-1lQQsvrV_Q2S?usp=drive_link)
